🚗 DriveSense – Smart Vehicle Maintenance Tracker

DriveSense is an offline Android application designed to help users track vehicle maintenance, fuel usage, and service schedules intelligently.
It goes beyond simple record-keeping by providing insights and predictions based on vehicle usage.

---

📌 Features

🚘 Vehicle Management

- Add and manage multiple vehicles (bike/car)
- Store basic details like name, number, and type
- Track current odometer reading

🛠️ Service Tracking

- Log service records with date, kilometer, cost, and notes
- View complete service history
- Get reminders for upcoming services

⛽ Fuel Logging

- Record fuel entries (liters, price, km)
- Maintain fuel history
- Track fuel expenses over time

📊 Smart Insights

- Calculate mileage (km/l)
- Predict next service based on usage
- Estimate days remaining for next maintenance
- Detect unusual drops in mileage

---

🧠 Core Logic

- Mileage Calculation:
  Distance traveled / Fuel consumed

- Service Reminder:
  Based on kilometer interval (e.g., every 3000 km)

- Prediction System:
  Uses average daily distance to estimate next service date

---

🛠️ Tech Stack

- Language: Java
- Platform: Android
- Database: SQLite (offline storage)
- UI: XML layouts

---

📱 App Structure

- Dashboard (overview & alerts)
- Add Vehicle Screen
- Vehicle Details Screen
- Service Log Screen
- Fuel Log Screen

---

🎯 Objective

To build a practical, offline-first mobile application that helps users manage vehicle maintenance efficiently while demonstrating real-world problem solving using Android development.

---


📌 Why DriveSense?

Unlike basic tracking apps, DriveSense focuses on insights and prediction, helping users make better decisions about vehicle maintenance instead of just storing data.
